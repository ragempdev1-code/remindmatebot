# RemindMate Bot

tg бот, считает дни до дня рождения, хранит заметки, шлет напоминания.

Можно держать сразу несколько ДР (свои и друзей), искать по заметкам, помечать их выполненными, редактировать и выгружать всё в txt. Напоминания умеют повторяться раз в день или раз в неделю. Все данные лежат в SQLite.

## Запуск

```bash
pip install -r requirements.txt
set BOT_TOKEN=твой_токен
python bot.py
```

Токен брать у [@BotFather](https://t.me/BotFather).

Если хочется поменять путь к базе:

```
set DB_PATH=remindmate.db
```
